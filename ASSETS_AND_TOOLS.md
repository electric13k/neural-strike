# FREE ASSETS & TOOLS FOR NEURAL STRIKE

## 1. AI Content Creation Tools (Free/Freemium)
- **3D Modeling**: [Meshy.ai](https://www.meshy.ai/) (Best for Godot-ready assets), [Luma AI Genie](https://lumalabs.ai/genie).
- **2D/Concept Art**: [Leonardo.ai](https://leonardo.ai/) (Great for tactical maps), [Canva Magic Media](https://www.canva.com/).
- **Sound Effects**: [MyEdit.online](https://myedit.online/en/audio-editor/ai-sfx-generator), [ElevenLabs SFX](https://elevenlabs.io/audio-native).
- **Music**: [Udio](https://www.udio.com/), [Suno](https://suno.com/).
- **Skyboxes**: [Blockade Labs Skybox AI](https://www.blockadelabs.com/) (Generate 360 HDRIs).

## 2. Royalty-Free Resources (CC0/Free)
- **3D Models**: [Kenney.nl](https://kenney.nl/) (Essential for FPS kits), [Poly.pizza](https://poly.pizza/).
- **SFX**: [FreeSound.org](https://freesound.org/), [Sonniss GDC Bundles](https://sonniss.com/gameaudiogdc).
- **Music**: [Incompetech](https://incompetech.com/) (Kevin MacLeod), [YouTube Audio Library](https://www.youtube.com/audiolibrary).
- **Textures**: [Poly Haven](https://polyhaven.com/textures), [AmbientCG](https://ambientcg.com/).

## 3. Development Tools
- **Engine**: [Godot Engine 4.3](https://godotengine.org/) (MIT License).
- **Visuals**: [Blender](https://www.blender.org/) (Modeling/VFX), [Krita](https://krita.org/) (Texturing).
- **Audio**: [Audacity](https://www.audacityteam.org/) (Cleaning up AI audio).
