## ⏱️ GENERATION WORKFLOW

### For Each Asset:

1. **ChatGPT (DALL-E 3):**
   - Paste prompt → Generate image
   - Download highest resolution
   - Save as `[asset_name]_concept.png`

2. **Hunyuan 3D (hy-3d.com):**
   - Upload concept image
   - Settings: High Quality, Dense Mesh, PBR ON
   - Generate (2-3 min)
   - Download GLB
   - Save as `[asset_name]_hd.glb`

3. **Tripo AI (Backup/Faster):**
   - Use Tripo prompt if image conversion fails
   - Generate (30-60 sec)
   - Download GLB
   - Save as `[asset_name]_tripo.glb`

---

## ✅ VERIFICATION CHECKLIST

Before importing to Godot, verify each model:

- ☑️ **Scale:** Robot is approximately same height as player model
- ☑️ **Pose:** T-pose with arms horizontal for rigging
- ☑️ **Proportions:** Humanoid (head, torso, 2 arms, 2 legs)
- ☑️ **Facing:** Front-facing (looking at camera)
- ☑️ **Details:** Panel lines, battle damage, sensors visible
- ☑️ **Clean mesh:** No floating parts or holes
