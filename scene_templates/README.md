# SCENE TEMPLATES

This folder contains **reference guides** for creating Godot scenes manually.

## Why Manual Creation?

`.tscn` files are Godot's binary scene format and cannot be easily generated as text. These templates provide:

1. **Complete node hierarchies** - What to create
2. **Exact property values** - What to configure  
3. **Step-by-step instructions** - How to build it

## Available Templates

### Phase 3: Level Environment
- **`PHASE3_MAIN_ARENA_SETUP.md`** - Detailed walkthrough for `main_arena.tscn`
- **`PHASE3_SCENE_QUICKSTART.gd`** - GDScript reference with all values

## How to Use

### Method 1: Follow the Markdown Guide (Recommended)
1. Open `PHASE3_MAIN_ARENA_SETUP.md`
2. Follow step-by-step in Godot Editor
3. Each step shows exactly what to create and configure
4. **Time: ~15-20 minutes**

### Method 2: Reference the GDScript
1. Open `PHASE3_SCENE_QUICKSTART.gd` in a text editor
2. See all node types, positions, and properties
3. Copy values into Godot Inspector as you build
4. **Time: ~10-15 minutes (if you know Godot)**
